## About

CA1B Connect is a section management platform I started for **Grade 11 Creative Arts and Technology (CA1B)** at Ateneo de Naga University.

I began building this on **June 11** as a personal idea to make our section more organized and easier to manage. Over time, it turned into a simple system that combines attendance, assignments, announcements, events, and student info in one place.

It’s not really a school project, more like something I’m building for our class to make things easier and more connected, while also experimenting with how far I can take a system like this.

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Future Plans

If this works well for CA1B, I want to expand it step by step:

### Phase 1 — CA1B (Current)

Make it stable and usable for our section first.

### Phase 2 — Senior High School (ADNU)

If it’s actually useful, expand it to other SHS sections in Ateneo de Naga University so each section can have its own space inside the system.

### Phase 3 — School-wide use

If it reaches that point, I’d like to explore if it can be used across the whole school for organizations, teachers, and students.

### Phase 4 — Bigger scale (maybe)

Long-term, I want to see if it can evolve into something bigger that schools can actually adopt, but that’s still far ahead.
